# Diabetes Data Analysis Project

## 📌 Overview
This project focuses on analyzing diabetes-related health data to identify important factors associated with diabetes risk and progression.  
The analysis combines **data preprocessing, statistical analysis, and data visualization** to uncover meaningful insights from the dataset.

The project aims to support better understanding of:
- Blood glucose behavior
- BMI distribution
- Insulin trends
- Blood pressure patterns
- Age-related diabetes risk

---

## 🎯 Project Objectives
- Clean and preprocess raw medical data
- Handle missing and invalid values
- Explore relationships between health indicators
- Visualize diabetes risk patterns
- Generate insights that may support early prediction and prevention strategies

---

## 📊 Dataset Features
The dataset includes several medical attributes such as:

- Pregnancies
- Glucose
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
- Outcome (Diabetes Diagnosis)

---

## ⚙️ Data Preprocessing
The preprocessing stage included:
- Removing invalid values
- Handling missing data
- Data transformation
- Feature categorization
- Preparing data for visualization and analysis

---

## 📈 Analysis & Visualizations
The project includes multiple visual dashboards and analytical charts, such as:

- Average glucose levels vs diabetes diagnosis
- BMI group distribution
- Diabetes risk classification
- Insulin levels by age
- Age group filtering
- Summary statistics dashboard

These visualizations help reveal patterns and trends associated with diabetes risk factors.

---

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Power BI

---

## 📌 Key Insights
- Higher glucose levels are strongly associated with diabetes diagnosis.
- Obesity represents the largest BMI category among diabetes cases.
- Insulin levels vary significantly across different age groups.
- Data visualization improves understanding of health-related risk patterns.
